## Project Overview & Context
This project was built as a guided challenge (inspired by Alex The Analyst) to master the core functionalities of Tableau Public. It provides a deep-dive analysis of the Airbnb rental market in Seattle, focusing on pricing strategies, geographical distribution, and revenue optimization based on zip codes and calendar data.

### Technical Skills Demonstrated:
* **Data Connection & Cleaning:** Managing data layers and addressing geographical location challenges.
* **Geographical Mapping:** Visualizing zip code areas, pricing densities, and handling edge cases in regional mappings.
* **Dynamic Interactivity:** Building interconnected sheets where filters apply seamlessly across maps, bar charts, and timeline views.

---

## Key Insights
* **Location Matters:** Higher average prices are heavily concentrated in specific waterfront and downtown zip codes.
* **Seasonal Trends:** Pricing and availability fluctuate significantly depending on the time of the year, showing clear peak seasons.
